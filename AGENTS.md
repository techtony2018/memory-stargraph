# Memory Stargraph agent contract

- Present every user-facing GBrain slug as a Markdown link whose label is the exact slug and whose target is `http://127.0.0.1:8788/?slug=<URL-encoded-slug>`.
- Browser automation must inspect existing tabs first, reuse a suitable same-origin or same-source tab, never close a reused user tab, and close only temporary tabs created by the current run.
- Preserve human control: do not auto-approve resolver proposals or perform destructive, privacy-sensitive, or irreversible operations without explicit authority.
